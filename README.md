# Therapist Website Starter

This repository contains a polished static starter website for a psychologist, psychotherapist, or cognitive behavioral therapist.

## Included

- `index.html` - main homepage
- `about.html` - therapist introduction page
- `services.html` - services and FAQ
- `contact.html` - contact and inquiry page
- `css/styles.css` - all shared styling
- `js/main.js` - reveal animations
- `assets/` - logo and favicon SVGs

## GitHub Pages

To publish:

1. Open the repository on GitHub.
2. Go to `Settings -> Pages`.
3. Under deployment source, choose the `main` branch and `/ (root)`.
4. Save and wait for the site URL to appear.

## Important

Replace all placeholder names, contact details, and copy before publishing the final version.
